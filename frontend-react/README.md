# TalkAI React Frontend

This is the React version of the TalkAI frontend, converted from vanilla JavaScript while maintaining the exact same design and functionality.

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will open at [http://localhost:3001](http://localhost:3001) (or the next available port).

Make sure your backend is running at `http://localhost:3000`.

### Building for Production

```bash
npm run build
```

## Features

- User authentication (Login/Signup)
- Real-time AI chat with streaming responses
- Chat management (create, rename, pin, delete)
- Profile management with avatar upload
- Dark mode support
- Settings page
- About page with FAQs

## Tech Stack

- React 18
- React Router DOM 6
- Context API for state management
- Vanilla CSS (same styles as original)
